# Introduction 
<I> Dr. Nim is a logic game based on a mechanical computer that can also serve as a good study object in computational or algorithmic thinking courses. </I>

#General rules of the game:

The game is played by 2 players, each taking turns removing 1, 2 or 3 marbles from a pile of a given number of marbles. Depending on the variation goal, the one taking the last marble is either the winner or the loser.

The design of the toy allows the game to be played by either 2 persons, a person against Dr. NIM or Dr. NIM against himself – this is configured through the turn switch, one of the moving plastic pieces.
